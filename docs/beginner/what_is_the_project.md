# O que e o projeto

O AIgenda e um projeto de agenda digital para empresas e equipes.

Ele foi pensado para ajudar pessoas a organizar compromissos, acompanhar atividades e centralizar informacoes importantes em um unico lugar.

Em vez de cada pessoa usar anotações soltas, planilhas ou mensagens espalhadas, o projeto reune tudo em um sistema mais organizado.

De forma simples, o AIgenda quer funcionar como uma agenda compartilhada com regras claras.

Cada empresa pode ter seu proprio espaco. Isso ajuda a manter os dados separados e evita confusoes entre equipes diferentes.

O projeto ainda esta em evolucao. Algumas partes ja existem como base de funcionamento e outras foram preparadas para crescimento futuro.
