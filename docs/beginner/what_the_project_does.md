# O que o projeto faz

O AIgenda ajuda a organizar a rotina de trabalho.

Hoje, a ideia principal do projeto e permitir que uma pessoa entre no sistema, acesse sua area e trabalhe com compromissos da agenda.

Na pratica, o projeto foi montado para fazer atividades como estas:

- permitir login de usuarios;
- mostrar uma area principal com atalhos importantes;
- exibir paginas de agenda, perfil e painel;
- cadastrar e consultar compromissos;
- separar os dados de cada empresa;
- preparar o sistema para crescer com novos recursos.

Mesmo quando alguma funcionalidade ainda esta simples na interface, a estrutura do projeto ja foi organizada para suportar evolucao.

No codigo, a parte tecnica desses compromissos aparece com o nome `appointments`, mas a ideia continua sendo a mesma: organizar itens da agenda.

Exemplo simples:

Uma empresa pode usar o sistema para controlar reunioes internas.

Cada usuario acessa sua conta, abre a area de compromissos e trabalha dentro do contexto da propria empresa.
