# Como o Sistema Funciona na Pratica

## Fluxo principal

```text
Entrar no sistema
  -> Criar ou editar compromisso
  -> Sistema valida regras
  -> Sistema salva com seguranca
  -> Usuario acompanha agenda atualizada
```

## Passo a passo simples

1. A pessoa faz login.
2. Ela tenta criar um compromisso.
3. O sistema verifica se o horario esta livre.
4. O sistema verifica se esse usuario tem permissao.
5. Se estiver tudo certo, o compromisso e salvo.
6. O sistema registra esse evento para controle interno.

## Exemplo do dia a dia

1. Maria entra no sistema.
2. Maria marca reuniao para 10:00.
3. O sistema percebe que Joao ja tinha reuniao nesse horario.
4. O sistema avisa o conflito.
5. Maria altera para 11:00.
6. Agora o sistema confirma e salva.

## O que isso traz de beneficio

- menos choque de horarios;
- menos retrabalho;
- mais confianca na agenda;
- mais organizacao para a equipe.
