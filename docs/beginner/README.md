# Documentacao para Iniciantes

Esta secao e voltada para quem esta conhecendo o AIgenda pela primeira vez. Os documentos aqui nao pressupõem experiencia tecnica aprofundada: o objetivo e dar uma visao clara do que o sistema faz, como ele funciona e como comecar a usa-lo.

| [glossary.md](glossary.md) | Significado dos termos tecnicos usados na documentacao |

## Por onde comecar

Se voce e novo no projeto, o caminho recomendado e:

1. Leia [overview.md](overview.md) para entender o proposito do sistema.
2. Consulte [features.md](features.md) para saber o que ja esta implementado.
3. Siga o [setup.md](setup.md) para subir o ambiente local.
4. Leia [workflow.md](workflow.md) para entender o fluxo de uso.
5. Consulte o [glossary.md](glossary.md) sempre que encontrar um termo desconhecido.
